the-brain integrates with AI assistants, IDEs, and remote servers.

## Supported Integrations

- **MCP Server** — Expose the-brain as a memory backend for Claude Desktop, Cursor, Zed
- **Remote Mode** — Run the-brain on a Linux server, connect from macOS
- **IDE Setup** — Auto-harvest from Cursor, Claude Code, Windsurf
- **Menu Bar App** — macOS menu bar for quick status and drag-drop ingest
