---
title: Environment Variables
description: Environment variable reference for the-brain
---

| Variable | Purpose | Default |
|----------|---------|---------|
| `THE_BRAIN_HOME` | Override config/data directory | `~/.the-brain` |
| `THE_BRAIN_CONFIG` | Override config file path | `~/.the-brain/config.json` |
| `THE_BRAIN_REMOTE_URL` | Remote server URL (agent mode) | — |
| `THE_BRAIN_AUTH_TOKEN` | Auth token for remote server | — |
| `THE_BRAIN_LOG_LEVEL` | Log level | `info` |
| `NO_MLX` | Disable MLX sidecar | unset |
| `THE_BRAIN_MLX_MODEL` | Override MLX base model path | `mlx-community/Llama-3.2-1B-Instruct-4bit` |
| `THE_BRAIN_MLX_DEPS` | Extra uv dependencies (semicolon-separated) | — |
| `LIBSQL_URL` | LibSQL/Turso URL | — |
| `LIBSQL_AUTH_TOKEN` | LibSQL/Turso auth token | — |
