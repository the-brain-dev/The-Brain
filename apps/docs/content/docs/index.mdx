# 🧠 the-brain

**Pluggable Cognitive Operating System for AI Agents** — an extensible background orchestrator that builds persistent memory from your AI coding assistant interactions.

---

## Why the-brain?

AI models forget everything between sessions. the-brain remembers — your coding patterns, preferences, corrections, and evolving style. It runs locally, costs nothing, and requires zero effort.

## Architecture at a Glance

```
Harvesters (Cursor, Claude Code, Windsurf, Gemini)
  ↓
⚡ INSTANT — Graph Memory (immediate corrections and preferences)
  ↓
⚖️ SELECTION — SPM Curator (filters noise from signal)
  ↓
🌌 DEEP — MLX LoRA Training (overnight consolidation)
```

## What the-brain Learns

- **Corrections** — "No, actually use `useCallback` here" → remembers for next time
- **Preferences** — "I prefer arrow functions" → injected into every prompt
- **Patterns** — Repeated tech choices → auto-suggests related concepts
- **Projects** — Per-project memory isolation with cross-project promotion
